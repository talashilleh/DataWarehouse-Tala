# Movie Rental Data Warehouse

## Project Description

This project presents a dimensional Data Warehouse design for a Movie Rental business using the Sakila OLTP database.

The project includes:

- Star Schema Design
- ETL Process
- SQL Data Warehouse Tables
- Analytical Queries
- Python Analysis using Pandas and Matplotlib
- Charts and Visualizations

---

# Technologies Used

- MySQL
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
